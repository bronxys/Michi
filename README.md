## 🔥 Instalación en Termux

> **Sigue estos pasos en orden:**

```bash
termux-setup-storage
```

```bash
apt update && apt upgrade && pkg install -y git nodejs ffmpeg imagemagick
```

```bash
git clone https://github.com/Ado-rgb/Michi-WaBot
```

```bash
cd Michi-WaBot
```

```bash
npm install
```

```bash
npm start
```
